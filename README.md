# GameRanker 🎮

GameRanker is a C++ program developed as a final university project. It allows users to manage, display, and explore game rankings through various sorting and filtering options.

## Features
* **Dynamic Sorting:** Sort the entire list of games by their ratings (scores).
* **Top 10 View:** Quickly filter and display only the best ten games in the ranking.
* **Flexible Display:** View game data in various ways based on user preferences.
* **Academic Project:** Created as a final assignment during the first year of university.

## Technical Details
* **Language:** C++
* **IDE:** Code::Blocks
* **Platform:** Windows Console

## How to Run
1. Download or clone this repository.
2. Open the `GameRanker.cbp` file in **Code::Blocks**.
3. Press `F9` (Build and Run) to start the application.
